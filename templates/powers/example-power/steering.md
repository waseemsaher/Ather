# Example Power Steering

> These instructions are injected into the agent system prompt when example-power is active.

## Purpose

This power provides tools and guidance for working with example files and demo content.

## Guidelines

- When creating examples, use clear variable names and add inline comments
- Prefer self-contained examples that run without external dependencies
- Always include expected output in code examples

## Available Tools

- `create_example` — Scaffold a new example file with boilerplate
- `validate_example` — Check that an example runs correctly
- `list_examples` — Show all examples in the current project
